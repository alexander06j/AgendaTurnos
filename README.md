# Turnos Microservice

Microservicio desarrollado con Spring Boot para la gestión de turnos médicos.

##  Tecnologías
- Java 25
- Spring Boot 4
- Spring Data JPA
- MySQL
- Maven

##  Funcionalidades
- Crear turnos
- Listar turnos
- Relación con pacientes

##  Configuración
1. Clonar repositorio
2. Configurar application.properties
3. Ejecutar:

```bash
mvn spring-boot:run
